Java Choices

There are many choices with which Java can branch out into, due to its high demand and its authentic design that is perfect for developing a wide ranges of applications and programs.

These choice include, but are not limited to:

- AI and machine learning

- Blockchain

- Web Services (e.g, Amazon)

- Mobile technologies (Android or iOS)

- Games & Mobile Applications

Because of how simple and authentic the design of Java is, it can branch out into many pieces of technology.
